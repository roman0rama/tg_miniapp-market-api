Welcome to Telegram Market API. Use next routes to get needed data from PostgreSQL:

User routes:

  • /api/users - get all users
  • /api/users/:id - add any id to get a specific user

    • Response example:

    {

    "id":35,
    "login":"useruser",
    "password":"$2b$10$JbFI2auSmdguxjY1h5GIACELt3BdaEgrAE7DKSAqxMc9i2tVqtUHO",
    "fullname":"Boris"

    }
  • /api/auth - post request for auth

    • Request example:

    {

    "login":"useruser",
    "password":"qwerty",

    }
  • /api/user - put request for update user data

    • Request example:

    {

    "id":1,
    "login":"useruser",
    "password":"qwerty123123",
    "fullname":"Boris"

    }
  • /api/user/:id - delete request for deleting user from database

    • Request example:

    {

    "id":"1",

    }

Reviews routes:

  • /api/reviews - get all reviews
  • /api/reviews - add new review
  • /api/reviews/:id - delete request. Add any id which exist in reviews table

Product cards routes:

  • /api/cards - get all cards
  • /api/cards/:id - add new review
  • /api/cards/:id - delete request. Add any id which exist in reviews table
  • /api/cards/:id - delete request. Add any id which exist in reviews table
